# ASR System for Call Center QA

Репозиторий с проектной документацией по MVP-системе автоматической обработки звонков колл-центра:

- автоматическая расшифровка аудио (ASR),
- диаризация (кто говорит),
- определение эмоциональной окраски реплик,
- агрегация метрик и приоритизация звонков для QA/супервизоров.

## О проекте

Цель проекта: сократить время ручного контроля качества звонков, повысить покрытие QA-проверки и ускорить поиск проблемных разговоров за счет ML-пайплайна ночной batch-обработки.

Планируемый результат MVP:

- обработка звонков за предыдущие сутки после ночной выгрузки,
- транскрипт с таймкодами и репликами,
- эмоциональная метка по репликам,
- агрегированные индексы по звонку и списки звонков для приоритизации QA-разбора.

## Текущий статус

- Этап: подготовка и формализация ML System Design документа для MVP-пилота.
